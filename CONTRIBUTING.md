# Contributing

Thank you for your interest in this project.

## Source-Available Notice

This repository is **source-available** under a proprietary license. You may view the code for reference and educational purposes, but you may not use, modify, or distribute it without prior written permission.

## Reporting Issues

If you find a bug or have a suggestion:

1. Open an issue using the appropriate template
2. Provide as much detail as possible
3. Include reproduction steps for bugs

## Feedback

General feedback and questions are welcome via GitHub Issues or Discussions.

## Licensing Inquiries

If you're interested in using any part of this codebase, please open an issue with the subject "Licensing Inquiry" and describe your intended use case.
